Named Credential — Flow HTTP Callout

{!$Credential.PricingAPI_NC}/prices/{!productCode} Method: GET Header: Authorization → Bearer mock-token-latam-2026 Timeout: 10000ms

Mapeamento Flow → Resposta da API

productName ← {!httpResponse.body.product_name} unitPrice ← {!httpResponse.body.unit_price} currencyCode ← {!httpResponse.body.currency} priceBRL ← {!httpResponse.body.price_brl} discountPct ← {!httpResponse.body.discount_reseller_pct} lastUpdated ← {!httpResponse.body.last_updated}
